## Syntax

```
NewRelic.shutdown()
```

Shut down the agent within the current application lifecycle during runtime

## Requirements

Agent version 6.10.0 or higher.

## Description

This API provides the ability to shutdown the agent within the current application lifecycle during runtime. Once the agent has shutdown, it cannot be restarted within the current application lifecycle.



## Examples

### Shut down the Android agent

Example of shutting down the agent:

```
NewRelic.shutdown()
```
