    Browser monitoring in New Relic provides a real user monitoring (RUM) solution. It measures speed and performance as your end users navigate to your site through different web browsers, devices, operating systems, and networks. But browser monitoring goes far beyond the providing information about the initial page load. Use it to measure full page life cycle data and start getting the info you to help ensure customer satisfaction.
